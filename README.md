# Legion

Legion 是 coding agent 与 IM 平台之间的桥接层。

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Type check
npm run typecheck

# Lint
npm run lint
npm run lint:fix

# Format
npm run format
npm run format:check

# Test
npm run test
npm run test:watch

# Build
npm run build
```
